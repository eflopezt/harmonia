Requisitos de conexión
  • El servidor de Harmoni y el reloj deben estar en la misma red local (o conectados via VPN).
  • El puerto 4370 debe estar habilitado en el firewall.
  • El reloj debe tener una IP estática (configurada desde el menú del dispositivo o via DHCP reservado en el router).
  • Usa TCP para dispositivos ZKTeco 2015 en adelante. UDP para modelos muy antiguos.

Tip: Configurar IP en ZKTeco
  1. En el reloj: Menú → Comm → Ethernet
  2. Configurar IP Address (fija)
  3. Configurar Subnet Mask y Gateway
  4. Reiniciar el dispositivo
