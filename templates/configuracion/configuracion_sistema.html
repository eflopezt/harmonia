Telegram Bot — Reclutamiento

Permite publicar ofertas de empleo automáticamente en un canal de Telegram. Crea un bot en @BotFather, agrega el bot como administrador del canal y pega el token aquí.

Obtenlo desde @BotFather → /newbot → copia el token de acceso.
@username para canales públicos, -100XXXXXXX para canales privados. El bot debe ser admin del canal.
{# ── WhatsApp Business Cloud API ── #}
WhatsApp Business — Reclutamiento

Permite enviar ofertas de empleo a número(s) de WhatsApp via Meta Cloud API. Requiere una cuenta de Meta Business con WhatsApp Cloud API habilitada y un número de teléfono de empresa verificado.

ID del número en Meta Developer Console → tu App → WhatsApp → Phone Numbers. No es el número de teléfono real.
Token permanente de la app. Usar System User Token (nunca token de usuario temporal). Obtener en: Meta Business Suite → Configuración → Usuarios del Sistema.
Formato internacional sin +: 51XXXXXXXXX. Separar por coma para múltiples. El número receptor debe haber iniciado conversación con el bot primero (ventana de 24h de Meta).
